# git教學

## git初始化 new repo

```bash
git config --global user.name "misawa3017"
```

```bash
git config --global user.name "misawa3017"
```

```bash
echo "# bashnote" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/misawa3017/bashnote.git
git push -u origin main
```
